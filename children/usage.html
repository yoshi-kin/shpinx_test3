3.1 Pythonソースコードからドキュメント作成#

3.1.1 ディレクトリ作成#

/sample_project
├── /docs
└── /src
├── __init__.py
└── sample_src.py

3.1.2 ソースからドキュメント作成#

$ cd sample_project
$ sphinx-apidoc -F -o docs/ src/
$ cd docs

3.1.3 設定ファイル#

conf.py

# コメントアウト解除
import os
import sys
sys.path.insert(0, os.path.abspath('../../src'))

# ...

extensions = [
    'sphinx.ext.autodoc',
    'sphinx.ext.viewcode',
    'sphinx.ext.napoleon',
    'sphinx.ext.todo',
]

# ...

html_theme = 'sphinx_book_theme'

src.rst

src package
===========

.. Submodules
.. ----------

sample\_src module
----------------------

.. automodule:: sample_src
    :members:
    :undoc-members:
    :show-inheritance:

.. Module contents
.. ---------------

.. .. automodule:: src
..    :members:
..    :undoc-members:
..    :show-inheritance:

$ make html

_build/html/index.htmlを開く。

3.1 作成したコンテンツからドキュメント作成#

3.1.1 作成#

  • プロジェクト名

  • ドキュメントの製作者

  • プロジェクトのバージョン

  • プロジェクトのロケール

  • プロジェクトディレクトリ

$ sphinx-quickstart -q -p プロジェクト名 -a 製作者 -v プロジェクトバージョン -l プロジェクトロケール プロジェクトディレクトリ
例）sphinx-quickstart -q -p "Book Review" -a sphinx-users.jp -v 1.0 -l ja project_dir

3.1.2 ディレクトリ構成#

/sample_project
├── _build
├── _static
├── _templates
├── conf.py
├── index.rst
├── make.bat
└── Makefile

3.1.3 HTML作成#

$ make html

index.rstをもとに_build/htmlが作成される。

3.1.4 HTML表示#

_build/html/index.htmlを開くと、ドキュメントのトップページが表示される。
